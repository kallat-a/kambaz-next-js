# Kambaz Next.js

A learning management system (LMS) built with Next.js, React, and TypeScript.

## Getting Started

```
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Deployment

- **Live Site:** [kambaz-next-js-ahaan-kallat.vercel.app](https://kambaz-next-js-ahaan-kallat.vercel.app)
- **GitHub:** [kallat-a/kambaz-next-js](https://github.com/kallat-a/kambaz-next-js)

## Project Structure

- `/app/labs` - Lab exercises and HTML examples
- `/app/(kambaz)` - Kambaz LMS application (Account, Dashboard, Courses, etc.)
